## [5.1.0](https://github.com/Theauxm/ChainSharp/releases/tag/5.1.0) (2026-02-10)

### Features

- Initial release with Railway Oriented Programming support
- Core workflow chaining with `Either` pattern
- Effect-based workflows with automatic metadata tracking
- Multiple data persistence providers (PostgreSQL, In-Memory)
- Mediator pattern integration for workflow discovery
- Hangfire job scheduling integration
- Comprehensive step isolation and testing support

- Getting started guide
- Core concepts documentation
- Architecture overview
- Usage examples
